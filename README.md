# Tarot Pro Lite

A soft feminine tarot-inspired Obsidian theme with lavender glow, moonlight gradients, mystical cards, and a free tarot dashboard.

![Screenshot](screenshot.png)

## Installation

1. Download this repository or the latest release.

2. Create this folder inside your vault:
.obsidian/themes/TarotProLite/

3. Put these files inside:
manifest.json
theme.css

4. Open Obsidian → Settings → Appearance → Themes
Select Tarot Pro Lite

## Included Free Dashboard

This repository also includes:
home-tarot.md

To use it:
1. Copy home-tarot.md into your vault root or into a folder like:
Dashboard/home-tarot.md

2. Open the note in Obsidian.
Rename it if you want, for example:
Home.md
Tarot Dashboard.md

3. Optionally install the Homepage plugin and set home-tarot.md as your startup page.
